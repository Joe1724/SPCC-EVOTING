# Nominee Seeder Instructions

## Available Seeders

### 1. NomineeSeeder (Default - UI Avatars)
Uses simple avatar placeholders with initials.

```bash
php artisan db:seed --class=NomineeSeeder
```

### 2. NomineeSeederWithRealPhotos (Realistic Photos)
Uses randomuser.me API to download realistic profile photos.

```bash
php artisan db:seed --class=NomineeSeederWithRealPhotos
```

## Image Storage

All nominee images are stored in:
```
storage/app/public/nominees/
```

Make sure you have run:
```bash
php artisan storage:link
```

## Resetting and Reseeding

To clear nominees and reseed:

```bash
# Clear all nominees
php artisan tinker
>>> App\Models\Nominee::truncate();
>>> exit

# Reseed with new data
php artisan db:seed --class=NomineeSeeder
```

Or use the full database reset:

```bash
php artisan migrate:fresh --seed
```

## Customizing Nominee Data

Edit the seeder files:
- `database/seeders/NomineeSeeder.php` - For UI Avatars version
- `database/seeders/NomineeSeederWithRealPhotos.php` - For realistic photos version

Each nominee requires:
- `first_name` - First name
- `last_name` - Last name
- `course` - Academic course (e.g., "BS Computer Science")
- `position` - Position running for
- `partylist` - Political party affiliation
- `description` - Campaign description/platform
- `image` - Photo path (auto-generated by seeder)

## Troubleshooting

**Images not showing?**
1. Make sure storage is linked: `php artisan storage:link`
2. Check file permissions on `storage/app/public/nominees/`
3. Verify images exist in the storage directory

**Seeder fails?**
1. Check internet connection (for downloading images)
2. Ensure `storage/app/public/` directory is writable
3. Try the basic NomineeSeeder if API fails

**Want to use your own images?**
1. Place images in `storage/app/public/nominees/`
2. Update the `image` field in the seeder to match your filenames
3. Run the seeder
